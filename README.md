# Simple CRUD API

## Description

Simple CRUD API using in-memory database underneath.

## Installation

```bash
git clone git@github.com:akrutsko/node-crud-api.git
cd node-crud-api
npm i
```

## Start the app

Development:

```bash
npm run start:dev
```

Production:

```bash
npm run start:prod
```

## Implementation details

1. Implemented endpoint `api/users`:
   - **GET** `api/users` is used to get all persons
